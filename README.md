# Viewgewind
Unsere Gruppe: Maurice F. (Functinal Programming), Jenni (Visual Programming) und Alex 

Geplant: 

Spieler 1: Beginnt das Spiel mit dem ersten Zug und wählt in eine Spalte aus, in dieser wird dann der Chip rein fallen bis zur untersten Position. 

Spieler 2: Dieser wiederholt dann den Ablauf von Spieler 1 nur mit der Möglichkeit das er an der gleichen Position seinen Chip setzen kann, wenn dies passiert wird sein Chip über den ersten liegen. 

Spielweise: 

Spieler wählt die jeweilige spalte aus in die der Chip fällt. Die Software prüft ob dir ein Chip liegen kann und schaut ob in den umliegenden Feldern noch andere Chips derselben Farbe liegen. Sind 4 vorhanden ist das Spiel beendet. 

Ziel des Spieles: 

Ziel des Spieles ist es in einer Reihe/Spalte oder in einer Diagonale 4 Chips seiner Farbe zu haben ohne, dass diese von dem anderen Spieler unterbrochen wird. Gewonnen hat der, der diese Anforderung erfüllt.
